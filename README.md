## 技术栈

- Vue 3 - 渐进式 JavaScript 框架
- Vite - 下一代前端构建工具
- Element Plus - 基于 Vue 3 的组件库
- Vue Router - 官方路由管理器
- Pinia - Vue 的状态管理库

## 功能特性

- 📱 响应式设计，支持移动端访问
- 🌓 支持亮色/暗色主题切换
- 📝 文章展示和管理
- 🎨 项目作品集展示
- 💭 动态分享功能
- 🔍 文章搜索功能
